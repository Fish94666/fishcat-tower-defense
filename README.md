# 魚貓塔防

這是一個用 HTML、CSS 和 JavaScript 製作的小型瀏覽器塔防遊戲。

目前版本包含：

- 兩種塔：`特務魚` 和 `狙擊魚`
- 共 8 波的關卡流程，`斥候怪` 和 `重裝怪` 的定位更清楚
- 選中塔之後可以直接升級或賣出，而且金額直接顯示在按鈕上
- HUD 已重新整理：手機版會優先顯示遊戲主畫面，生命與金錢獨立分區，波數與即時訊息顯示在地圖上方

## 本機遊玩

直接用瀏覽器開啟 `index.html` 即可。

## 網頁版遊玩

如果你想直接打開已部署的網頁版，請前往：

[https://fish94666.github.io/fishcat-tower-defense/](https://fish94666.github.io/fishcat-tower-defense/)

如果網站內容看起來沒有更新，可以先嘗試重新整理，或使用無痕視窗重新打開。

## HUD 怎麼看

- `生命` 和 `金錢` 會顯示在操作區的資源卡裡。
- `波數` 和目前的即時提示會顯示在地圖上方，方便邊看戰場邊掌握進度。
- 選到塔之後會打開精簡資訊卡，目前會顯示塔名、等級、`傷害`、`射速`，以及 `升級` / `賣出` 按鈕。
